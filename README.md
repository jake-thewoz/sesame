## Quick Start

### 1. Installation

**Build from source (Rust required):**

```bash
git clone https://github.com/jake-thewoz/sesame.git
cd sesame
cargo build --release
./target/release/sesame --help
```

### 2. Create your first vault

```bash
sesame init
```

### 3. Add a password

```bash
sesame add
(follow prompts)
```

### 4. View your vault items

```bash
sesame list
```

### 5. Retrieve a password

```bash
sesame show <id>
```
